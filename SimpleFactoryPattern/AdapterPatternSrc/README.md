我们有一个MediaPlayer接口和一个实现MediaOlayer接口的实体类AudioPlayer。默认情况下AudioPlayer可以播放MP3格式的音频音乐。
还有一个接口AdvancedMediaPlayer 和实现AdvancedMediaPlayer 接口的实体类。该类可以播放vlc和MP4格式的文件
我们想要让AudioPlayer 播放其他格式的音频文件，为了实现这个功能，我们需要创建一个试下MediaPlayer接口的适配器类MediaAdapter，并使用AdvancedMediaPlayer对象来播放所需的格式。
AudioPlayer使用适配器类MeidaAdapter传递所需要的音频类型，不知道能播放所需的实际格式。
AdapterPatternDemo，我们演示类使用AudioPlayer类来播放各种格式。
